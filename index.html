Game Instruction


To START the game press button and set how long a sequence you want to guess.

Press button and wait for the sequence to end.

Now your turn!

Repeat the sequence by pressing the color buttons.

If you are correct, the game displays the V sign in the highest score counter at the top and responses with a "success" sound, you can continue by adding more colors to a sequence using button.

Remember! You can not move back! You can only add colors.

If you fail to repeat it in the exact order, the game displays the X sign, stops and zeroes the main counter immediately.
You need to start a new game.

You can add as many colors as you wish (the limit is 99).

During your turn, you can use one of your life points by pressing button (the start amount is three 3) to see the last sequence again.

Press button anytime you want to stop the game.

Remember! When you press , the main counter zeroes and you need to start from the beginning.

The number at the top is your highest score.

The number at the center among the color buttons is your current sequence length.
